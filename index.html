PingMe

A MERN stack application that aggregates web-toon/animation updates and notifies users based on their subscriptions. My current role in the team is writing and testing functions that gather latest updates from various API and websites and query the results into a noSQL database.

PeerConnect

PeerConnect is an open-source server implementation tool that aims to serve static assets (videos/images) over a peer-to-peer delivery network powered by WebRTC, WebTorrent, and WebSocket to lessen server load for media heavy web applications and websites.

Pomodoro

A configurable timer application built using React. Utilizes React lifecycle methods to handle asynchronous events in order to manage state. Web workers run Javascript in the background without interfering with the user interface even when the browser is idle.
